Detailed Description

Random spectrum combination.

(GUI name: "")

#include "sw_sdk/effects/burrito.hpp"

Copies side-channel data to the input at random locations.

Mode

Combination mode.

( Replace )( Sum )

(GUI name: "Key")

(GUI name: "Direction")

Minimum<0>, Maximum<100>, Default<50>, Unit<'bw%'>
  • Replace: input is replaced by the side-channel at random locations.
  • Sum: input is summed with side-channel at random locations.
Range

Maximum amount of spectrum to randomize.

(in %bw, percentage of bandwidth i.e. total frequency range)

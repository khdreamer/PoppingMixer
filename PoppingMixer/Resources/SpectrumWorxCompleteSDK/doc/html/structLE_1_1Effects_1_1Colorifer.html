Detailed Description

Spectral colouring.

(GUI name: "")

#include "sw_sdk/effects/colorifer.hpp"

Transfers the frequency shape from the side-channel to the signal arriving through the main input. It calculates main/side over the selected bandwidth, and then applies this "colour" to main. There is an option to also replace phases of the main channel with those from side.

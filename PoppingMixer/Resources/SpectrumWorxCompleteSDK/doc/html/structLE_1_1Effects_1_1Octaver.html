Detailed Description

Adds two octaves.

(GUI name: "")

#include "sw_sdk/effects/octaver.hpp"

This module adds two (or one) octaves to the input signal. Range is up or down two octaves. Each octave has Gain control and cut-off frequency is shared between both octaves.

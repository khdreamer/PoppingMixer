Audio IO

What is audio IO?

Audio IO is a simple audio input/output library which enables the user to easily capture audio from a microphone or read it from a file as well as to send audio to speakers/headphones or to a file. It easily integrates with core Little Endian sound effect and analysis SDKs, as well as with related native OS facilities. This library aims to simplify the programmer's task of capturing sound from an input before the sound manipulation is done and of sending the transformed sound to an output device or file afterwards.

Detailed Description

Provides synchronized/blocking access to a Device instance.

Useful mostly for single-threaded command line applications.

Constructor & Destructor Documentation

BlockingDevice ( Device const &  )

Effect: Creates the blocking wrapper for the given Device object which must outlive the created BlockingDevice object.

Member Function Documentation

void startAndWait ( )

Effect: Starts the associated Device instance and blocks until the stop() member function is called (from another thread).
Preconditions: The same as for Device::start().

void stop ( )

Effect: Stops the associated Device and unblocks the thread blocked by the call to startAndWait().
Preconditions: A prior call to startAndWait().

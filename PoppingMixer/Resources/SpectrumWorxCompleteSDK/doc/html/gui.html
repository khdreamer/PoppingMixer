The GUI tool

Designing sounds using the SpectrumWorx VST/AU plugin

This section will explain how to design desired sound effects using a GUI tool.

Introduction

The technology behind the SpectrumWorx SDK is the same technology we use to power our popular SpectrumWorx VST/AU plugin. As a convenience, this plugin can be used as a GUI tool for easier experimentation and exploration of the effects and soundscapes that are achievable with the SpectrumWorx SDK. It comes with a large collection of presets and allows creation of custom ones thereby making the sound effects design an easier task. A full commercial licence for the SpectrumWorx VST/AU plugin is awarded to every SpectrumWorx SDK customer free of charge.

The SpectrumWorx VST/AU plug-in
Note
As a plugin, SpectrumWorx can only be used inside a DAW (Digital Audio Workstation) host that supports the VST2.4 and/or AU plug-in formats (almost all of them support at least one or the other).

Download

The SpectrumWorx VST/AU plug-in can be downloaded from http://littleendian.com/consumers, it supports Microsoft Windows and Apple OS X operating systems.

Digital audio workstations (VST/AU hosts)

There are many available DAW hosts (Ableton Live, PreSonus StudioOne, Audacity (free), Steinberg Cubase, Adobe Audition, Logic, Sound Forge, etc.). However, for ease of installation, easy of use, flexible licensing, we recommend:

  • REAPER (Windows & OS X), http://www.reaper.fm/download.php
  • a short tutorial on how to use the SpectrumWorx plugin as a GUI tool inside REAPER: REAPER tutorial.

Workflow

The basic workflow should be the following:

  • install a DAW host (Reaper or any other)
  • install the SpectrumWorx plugin (http://littleendian.com/consumers)
  • start the DAW host
  • insert/create an audio track
  • add SpectrumWorx to the track via "insert VST/AU"
  • use the GUI to design desired sound effects
  • save sound effect presets to SpectrumWorx preset files (simple XML files with the .swp extension)
  • transfer/transform the settings from the preset file into SpectrumWorx SDK based C++ code.
Note
  • The SpectrumWorx SDK API and the SpectrumWorx VST/AU plugin GUI interface do not exactly match:
    • names of some effects and parameters are different, this is clearly stated in the documentation for each effect and/or parameter when there is a difference between the SDK API and the GUI interface
    • value representations for some parameters also differ between the SDK API and the GUI interface, for example the GUI might display a gain parameter's value in decibels while the SDK might use a linear value (note that the values saved into the .swp preset files will always use the plugin's internal 'format' which is the same as the SDK 'format' so the parameter values found in SpectrumWorx presets need no transformation and can be directly transfered to SpectrumWorx SDK based C++ code)
    • the SpectrumWorx SDK does not provide LFO functionality
    • the SpectrumWorx SDK only provides one global gain parameter (which maps to the Output gain parameter in the plugin).
  • For a detailed guide on how to use the GUI plug-in inside REAPER see: REAPER tutorial.

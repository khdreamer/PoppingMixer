Member Function Documentation

error_msg_t create ( char const *  pathToFile,
unsigned int  numberOfChannels,
unsigned int  sampleRate 
)

Effect: Creates a file at pathToFile for writing.
The function can be called many times, i.e. one instance of OutputWaveFile can be used to create and write as many WAVE files as desired. Note that the function offers only the basic (error safety) guarantee.
Postconditions: Unless an error is reported, the file was successfully created with preallocated space for the header/metadata.

error_msg_t write ( float const *  pInput,
unsigned int  numberOfSampleFrames 
)

Effect: Writes interleaved numberOfSampleFrames * numberOfChannels samples from pInput into the underlying file.
Preconditions:

  • a successful create() call
  • the buffer pointed to by pInput must hold at least numberOfSampleFrames * numberOfChannels samples.

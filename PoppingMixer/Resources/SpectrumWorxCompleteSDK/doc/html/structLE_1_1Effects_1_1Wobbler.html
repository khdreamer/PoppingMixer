Detailed Description

Input spectrum amplitude sine modulation.

(GUI name: "")

#include "sw_sdk/effects/wobbler.hpp"

This module creates a sinusoidal-based “wobbling” sound by changing the amplitudes of the input frequencies.

Member Typedef Documentation

Amplitude

Intensity of the wobbling (or how much the sound seems to go up and down).

(GUI name: "Key")

Minimum<0>, Maximum<+48>, Default<6>, Unit<'dB'>
Period

Rate of wobbling.

(GUI name: "Direction")

Minimum<10>, Maximum<1000>, Default<100>, Unit<'ms'>
PreGain

Shifts the whole input before the wobbling stage.

(GUI name: "1")

MaximumOffset<48>, Unit<'dB'>

It might be advisable to start with a negative PreGain value that reflects the positive setting of the Amplitude setting.

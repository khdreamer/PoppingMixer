Member Function Documentation

error_msg_t open ( char const *  pathToFile)

Effect: Opens the file pointed to by pathToFile for reading.
The function can be called many times, i.e. one instance of InputWaveFile can be used to read as many WAVE files as desired. Note that the function offers only the basic (error safety) guarantee.
Postconditions: Unless an error is reported, the file was successfully opened and its header parsed making it ready for audio and metadata extraction.

void read ( float *  pOutput,
unsigned int  numberOfSampleFrames 
) const
inherited

Effect: Reads interleaved numberOfSampleFrames * numberOfChannels() samples into pOutput and adjusts the current file position (as reported by remainingSamples()).
Preconditions:

  • a successful open() call
  • remainingSamples() >= samples
  • the buffer pointed to by pOutput must be large enough to hold numberOfSampleFrames * numberOfChannels() samples.

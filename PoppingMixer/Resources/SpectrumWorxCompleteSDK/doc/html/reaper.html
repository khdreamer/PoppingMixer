How to use the SpectrumWorx GUI inside REAPER

SpectrumWorx is a plug-in for digital audio workstations (DAWs), and it can only be used inside such DAW software. Here we present a step-by-step tutorial on how to use SpectrumWorx with REAPER as a lightweight and user-friendly DAW.

Step 1: Get REAPER (digital audio workstation)

a. Download REAPER demo (size 5-10 MB) from http://www.reaper.fm/download.php.
b. Install REAPER with default options. Don't start REAPER yet.

Step 2: Get SpectrumWorx (plug-in for digital audio workstations)

c. Download the SpectrumWorx demo (size 3-5 MB) from http://www.littleendian.com/download/.
d. Install SpectrumWorx with default options.

Step 3: Start REAPER

e. Start REAPER. (Figure 1)

Figure 1: REAPER started

f. Insert an audio sample into REAPER: choose "Insert/Media file..." on the menu. (Figure 2)

Figure 2: REAPER with audio track inserted

Step 4: Insert SpectrumWorx into REAPER

g. Press "FX" icon to open the effects window. (Figure 2)
h. Select "VST" on the left window side. (Figure 3)

Figure 3: REAPER with FX window

i. Select "SpectrumWorx (Little Endian)" on the right window side. (Figure 3) If there is no SpectrumWorx on the list, please see Note below.
j. Press "OK". (Figure 3)

Step 5: Play the sample in REAPER

k. Play the sample: press play icon. (Figure 2)
l. Loop the sample: press loop icon. (Figure 2)

Step 6: Test SpectrumWorx

m. Press "Presets" knob on the SpectrumWorx main window. (Figure 4)

Figure 4: REAPER with SpectrumWorx inserted as FX

n. Select a preset in the Presets window. (Figure 5)

Figure 5: REAPER with SpectrumWorx, preset selected

o. Experiment with the knobs!

Note
Windows users: In case REAPER does not "see" the plug-in, please check the VST plug-in location path. This is where REAPER should look for SpectrumWorx.dll file.
Figure 6: REAPER plug-in path

What next?

We advise the user to continue with the following:

  • Study the Users's Guide which comes with the GUI tool.
  • Design sound in the GUI.
  • Open preset files (.swp) in text editor, and transfer settings to the SDK code.

Detailed Description

Replaces main with side channel based on magnitude comparisons.

(GUI name: "")

#include "sw_sdk/effects/ethereal.hpp"

Compares the side-channel signal with that of the input and it replaces the input with the side signal if certain conditions are met: (input - side > or < threshold). Can replace magnitudes or phases, or both.

Member Typedef Documentation

Mode

Specifies what is to be replaced.

(GUI name: "1")

  • Both : replace both magnitudes and phases.
  • Magnitudes: replace only magnitudes.
  • Phases : replace only phases.
